Tippo.js

Tippo is simple UI only tool tip utility to help guide users on how to navigate and use your webpage! This is a Vanilla JavaScript module that create, append, hide tooltips attached next the element of interest. All capabilities are strictly UI based, no database required.

Tippo currently offers:

  • 5 color themes

  • Positioning relative to the target:

  • Entry animations such as fade in

  • Reminding animations to help users see the tool tip

  • LocalStorage persist state based on the tool tip's ID
