Tippo.js

Github / Documentation

Tippo is a simple UI only tool tip utility to help guide users on how to navigate and use your webpage! This is a Vanilla JavaScript module that can append tooltips attached next the element of interest. All capabilities are strictly UI based, no database required.

Tippo currently offers:

  • 5 color themes

  • Positioning relative to the target

  • Several entry animations for an immersive user experience

  • Reminder animations to help users see the tool tip

  • Hiding tool tip on future page visits after acknowledgement
